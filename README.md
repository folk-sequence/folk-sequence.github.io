# folk-sequence.github.io

Companion essays for the [Folk Sequence](https://www.youtube.com/@FolkSequence) YouTube series — daily music-production sessions in Bitwig Studio, recorded in continuous takes with no narration, no edits, no cuts.

Each episode ships with a long-form essay that traces a single idea from first principles. Essays live here as one Jekyll-rendered markdown file per episode at `NNN-slug.md`, published at `https://folk-sequence.github.io/NNN-slug/`.

- **YouTube**: https://www.youtube.com/@FolkSequence
- **jalopy.music artist page**: https://jalopy.music/folk-sequence

Managed by the [folk-sequence pipeline](https://github.com/bedwards/folk-sequence).
